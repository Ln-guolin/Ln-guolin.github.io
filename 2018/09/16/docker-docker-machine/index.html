前言

你可以用docker machine来做什么？
1，在mac和windows平台上安装和运行docker
2，提供和管理多个远程docker主机
3，提供swarm集群管理

安装

安装很简单，直接从github下载编译好的二进制文件即可，也可以参考官网详细介绍
下载地址：https://github.com/docker/machine/releases

Linux
Mac

mac客户端已经包含了machine，其实无需单独安装。

查看machine安装版本，可以正常输出版本信息即可。
