前言

docker-compose是编排容器的利器，特别是涉及多个容器的运行时，不需要敲各个容器的启动参数，环境变量，容器命名，指定不同容器的链接参数等等一系列的操作，通过docker-compose，它可以轻松完成容器编排，并且提供了scale扩容功能。
