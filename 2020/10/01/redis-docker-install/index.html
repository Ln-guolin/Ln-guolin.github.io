前言

    如果非要问怎么安装Redis最省心、最简单，那肯定要属Docker镜像安装了。

    docker hub镜像仓库中，有无数的开发者贡献了无数的镜像，可供选择，只需要一行docker命令，即可快速完成部署。不过要注意的是，最好选择带有OFFICIAL IMAGE标志的官方镜像，安全可靠！
